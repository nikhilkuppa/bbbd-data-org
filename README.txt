BBBD Data Processing Scripts
=============================

STRUCTURE
---------
bids/raw/       - Scripts to write raw BIDS-formatted data (experiments 1-5)
bids/derived/   - Scripts to write derived BIDS-formatted data (experiments 1-5)
matrix/         - Scripts to write matrix-format data files (experiments 1-5)
config/         - Reference files bundled with the scripts (channel locs, demographics, etc.)
bbbd_run_all.m  - Runs all scripts in sequence
config.m        - Central configuration file (edit this before running)

SETUP
-----
Open config.m and set the five paths at the top:

  data_dir        - root folder containing experiment_1, experiment_2, ... subfolders
                    each subfolder must have: raw/, processed/, metadata/ subdirectories
  output_dir      - folder where all BBBD output will be written
  bbbd_source_dir - path to the BBBD-unzipped reference directory (the pre-existing BIDS
                    release containing experiment1/, experiment2/, ... subdirectories).
                    Used only to copy README.md into each output experiment folder.
  eeglab_path     - path to EEGLAB toolbox directory
  biosig_path     - path to BioSig toolbox directory (used by bbbd_123_derived_continuous only)

NOTE: config/ already contains BioSemi64.mat, doIntervention_indexing.mat,
experiment4_demographic.mat, and experiment5_demographic.mat.
You must manually place int_metadata.mat (the combined participant metadata file)
at config/int_metadata.mat before running experiments 4 and 5. See ABOUT INT_METADATA
section below.

HOW TO RUN
----------
1. Open MATLAB and set the working directory to this folder (data_code/).
2. Edit config.m with your paths.
3. Ensure config/int_metadata.mat is in place (see below).
4. Run bbbd_run_all.m to execute all scripts in order, or run individual
   scripts directly as needed.

Scripts run in the following order per experiment group:
  0. bbbd_generate_bids_descriptors.m  (participants, phenotype files, dataset_description)
  1. raw_eye_heart                     (physiological + eyetracking raw)
  2. raw_eeg                           (EEG raw)
  3. derived_continuous                (continuous derived signals)
  4. derived_discrete                  (discrete event timestamps)

WHAT THE SCRIPTS GENERATE
--------------------------
Per experiment (experiment1/ through experiment5/ under output_dir):

  Root level:
    dataset_description.json    generated from hard-coded metadata
    participants.tsv             generated from Experiment_N_demographics.mat
    participants.json            generated alongside participants.tsv
    .bidsignore                  generated (same content for all experiments)
    README.md                    copied from bbbd_source_dir/experimentN/README.md

  phenotype/ :
    stimuli_questionnaire_scores.tsv    generated from Experiment_N_Points.mat
    stimuli_feedback_meta.tsv/.json     generated from questionnaire.mat (exp2-5)
    asrs_questionnaire.tsv/.json        generated from questionnaire.mat (exp2-5)
                                          per-question q1-q18 for exp2/3; summary for exp4/5
    digit_span_scores.tsv/.json         generated from digitspan.mat (exp2-5)

  NOT generated (copy manually from BBBD-unzipped if needed):
    phenotype/stimuli_questionnaire.tsv   per-question individual answers with correct answers
    phenotype/stimuli_questionnaire.json  question text lookup

  sub-*/ses-*/ :
    All raw and derived BIDS data files are generated by the bids/ scripts.

ABOUT INT_METADATA.MAT
-----------------------
int_metadata.mat is a combined struct array containing processed per-participant data
for all experiments 4 and 5 participants. It is NOT simply the individual metadata files
from experiment_4/metadata/ — it is a pre-combined file that includes processed signal
segments (ECG r-peaks, breath timestamps, eye events, etc.) organized as:

  metadata_full(i).participant_no      participant number
  metadata_full(i).segments(s,v)       processed data per stimulus s, session v
    .hr.timestamps_rel_stim            R-peak timestamps
    .respiration.timestamps_rel_stim   breath timestamps
    .blinks / .saccades / .fixations   eye event timestamps
    .eye / .pupil                      interpolation timestamps
  metadata_full(i).(other fields)      additional participant metadata

This file is generated by a separate preprocessing pipeline. If you have the original
raw experiment_4 processed .mat files (data_eeg_processed.mat, data_ecg.mat, etc.)
you can reconstruct it by running the preprocessing scripts that extract per-participant
segments and combine them. Once created, place it at config/int_metadata.mat.

DEPENDENCIES
------------
- EEGLAB (eeglab2024.2 or compatible)
- BioSig toolbox (required by bbbd_123_derived_continuous.m only)
